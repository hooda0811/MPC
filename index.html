Petunjuk Penggunaan

Berikut merupakan petunjuk penggunaan dashboard MPC.

Alur singkat
Operator buat WO → Maintenance atur schedule/permit → Vendor kerjakan → Done → Validation operator → Approved → Maintenance cek akhir → Close.
Review
WO baru masuk, belum diproses.
Open
WO sudah siap dikerjakan.
Progress
Pekerjaan sedang berjalan.
Done
Vendor selesai kerja, lanjut validasi operator.
Validation
Menunggu operator cek hasil vendor.
Approved
Operator sudah approve, menunggu close dari maintenance.
Close
WO selesai dan ditutup.
